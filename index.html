Introducción a JSON

¿Qué es JSON?

JSON, que significa JavaScript Object Notation, es un formato ligero para el intercambio de datos. Es fácil de leer y escribir para humanos, y fácil de analizar y generar para máquinas.

Es independiente del lenguaje y se utiliza comúnmente en la programación web para el intercambio de datos entre un servidor y una aplicación web o móvil.

Estructura y Tipos de Datos

  • Objetos: Conjunto de pares clave-valor encerrados entre llaves {}.
  • Arrays: Lista ordenada de valores encerrados entre corchetes [].
  • Tipos de Datos: Números, cadenas de texto, booleanos, null.

Ejemplo de JSON

{
  "titulo": "El Principito",
  "autor": "Antoine de Saint-Exupéry",
  "publicado": 1943,
  "idiomas": ["francés", "inglés", "español"],
  "disponible": true
}
